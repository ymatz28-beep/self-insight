## Self-Insight 推奨サブスクモデル

### Tier 1: 無料（生年月日のみ）
- **含む**: 四柱推命・九星気学・六星占術の基本プロファイル、2026年運勢概要、月間運勢（当月のみ）
- **目的**: 「鳥肌が立つほど正確」な衝撃体験で掴む
- **制限**: 性格分析なし、Cross Analysisなし、過去月/未来月ブラー

### Tier 2: ¥480/月 or ¥4,800/年（+性格診断）

- **追加**: Cross Analysis（占術×性格の掛け合わせ）
- **目的**: 自己理解の深さに価値を感じるユーザー

### Tier 3: ¥980/月 or ¥7,800/年（全開放+月次更新）
- **追加**: 月次パーソナライズドガイダンス更新（毎月1日に自動更新）

- **追加**: ジャーナル機能（運気×日記の掛け合わせ振り返り）

## チャーン防止 5メカニズム

### 1. 運気カレンダー（時間軸ロック）
- **仕組み**: 毎月1日に新しい月間ガイダンスが解放される
- **心理**: 「来月の運勢が気になる」→ 解約できない
- **競合参考**: ゲッターズ飯田（毎朝の運勢チェック習慣化）

### 2. ジャーナル蓄積（データロックイン）
- **仕組み**: 運気データと連動した日記。過去の振り返りが可能
- **心理**: 「3ヶ月分のジャーナルがここにある」→ 離脱=データ喪失
- **競合参考**: CHANI（ジャーナル+月のサイクル）

### 3. 深度ロック（段階的開示）
- **仕組み**: Tier 1で核心を突く → もっと知りたい → Tier 2/3
- **心理**: 「ここまで分かるならフル版はどれほど…」
- **競合参考**: 16Personalities（無料で衝撃 → プレミアムで深堀り）

### 4. 相性機能（ソーシャルロック）
- **仕組み**: パートナーや友人を招待 → 二人の相性分析
- **心理**: 「二人のデータがあるから」→ 解約しにくい
- **競合参考**: Co-Star（友達との星座比較がコア機能）

### 5. 無料セーフティネット
- **仕組み**: 解約しても基本データは残る（閲覧はTier 1相当に制限）
- **心理**: 「戻ってきやすい」→ 再課金率向上
- **競合参考**: Netflix/Spotify（解約→再入会のハードル低い設計）

