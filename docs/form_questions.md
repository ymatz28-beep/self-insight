#### Mini-IPIP スコアリング

```python
# 逆転処理
def reverse(score):
    return 6 - score

# 因子別スコア（各4〜20点）
extraversion      = Q22 + reverse(Q23) + Q24 + reverse(Q25)
agreeableness     = reverse(Q26) + Q27 + reverse(Q28) + Q29
conscientiousness = Q30 + reverse(Q31) + Q32 + reverse(Q33)
neuroticism       = Q34 + reverse(Q35) + Q36 + reverse(Q37)
openness          = Q38 + reverse(Q39) + Q40 + reverse(Q41)

# 各因子を100点満点に正規化
def normalize(raw, min_val=4, max_val=20):
    return round((raw - min_val) / (max_val - min_val) * 100)

# 判定目安（正規化後）
#   75〜100: 高い
#   40〜74:  中程度
#   0〜39:   低い
```

### プログレッシブ離脱対応

Google Forms は途中保存ができないため、以下の方針で対応:

1. **Tier 1のみでも価値がある設計**: Tier 1完了時点で占術プロファイルが生成可能
2. **各Tier間に離脱ポイント（プレビュー画面）を設置**: ユーザーが続行するか判断できる
3. **所要時間の明示**: 各Tierの冒頭に所要時間目安を表示

### リニアスケールの表記

Google Formsのリニアスケールでは両端のラベルのみ設定可能:

- **HSP / Mini-IPIP**: 左端「全く当てはまらない」〜 右端「よく当てはまる」
- **ASRS**: 左端「全くない」〜 右端「非常に頻繁にある」
