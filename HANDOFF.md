# Self-Insight — HANDOFF

## Last Updated
2026-03-20

## Project Overview
複数の占術（四柱推命・九星気学・六星占術・西洋占星術・干支）+ 心理学ツール（エニアグラム・HSP/ADHD特性・独自SF/MBTI代替）を統合した、徹底的にパーソナライズされた自己理解ダッシュボードサービス。有料SaaS展開を目指す。

### Core Concept
- 「占い」ではなく「AI Self-Insight」— データ駆動のパーソナル・インサイト
- **統合インサイト first**: 個別占術の結果より先に、全システムを統合した「あなたはこういう人」を提示
- 年間運気予測（九星気学×六星占術の運気カーブ重ね表示）、健康/仕事/お金の3軸指針
- **過去の振り返り + 長期フォーキャスト**: 良い時・悪い時をフィーチャーし、具体的アクション提案
- Yumaの設計思想（WWH, 3S, Constancy, MECE, 7つの習慣, PDCA）をフレームワークに組み込み

### Why（3つのWhy）
- **個人的Why**: 自分がユーザー1号。identity.mdの延長で自己理解を深化
- **市場的Why**: 占い市場997億円。「複数占術 × 心理学 × ダッシュボード可視化」の統合サービスは空白
- **タイミングWhy**: 生成AIで占術アルゴリズム化コストが激減。Co-Star（占星術×AI）が500万DL

### Phase Plan
- **Phase 0**: Yuma自身のダッシュボードとして構築（identity.mdの延長）
- **Phase 1**: 友人5人にβテスト（無料）→ PMF検証
- **Phase 2**: 月額980円でローンチ

### SaaS UXフロー（Phase 2イメージ）
1. **入力ページ**: 生年月日・出生時刻（任意）・出生地 → 血液型
2. **性格診断**: 独自SF/MBTI/エニアグラム代替の質問セット → HSP/ADHD簡易判定
3. **結果出力**: ダッシュボード or PDFレポート

### Key Design Decisions
- MBTI/ストレングスファインダーは商標制約あり → **独自診断フレームワーク開発**
- 占術ロジック: OSSライブラリ活用（四柱推命: lunisolar系、西洋占星術: Swiss Ephemeris）
- **占術計算はLLM記憶に依存しない** — OSSライブラリ or 権威ある書籍からの実装 + ユニットテスト必須
- ブランディング: 「占い」と呼ばない。「パーソナル・インサイト」「自己理解ダッシュボード」
- 健康/仕事/お金の分岐は既存PJ（health-tracker, stock-analyzer, wealth-strategy）と接続
- **姓名判断**: オプションモジュール（日本語圏限定、ユニバーサル性なし）
- **レジリエンシー設計**: 各占術モジュールは独立。欠損データがあっても他モジュールの精度に影響しない

### レジリエンシー Tier設計
| Tier | 必要データ | 精度 | 有効モジュール |
|---|---|---|---|
| Tier 1（最小）| 生年月日のみ | 60% | 九星気学 + 六星占術 + 西洋占星術 + 干支 + 四柱推命(3柱) |
| Tier 2（標準）| + 血液型 + 出生地 | 70% | + 血液型インサイト |
| Tier 3（拡張）| + 性格診断回答 | 85% | + 独自SF/MBTI/エニアグラム + HSP/ADHD判定 |
| Tier 4（完全）| + 出生時刻 + 姓名 | 95% | + 時柱 + アセンダント + 月星座 + 姓名判断 |

### CX設計原則
- **統合インサイト first**: 個別システムの結果より先に、全占術を統合した人物像を提示
- **二人称（あなたは〜）**: Co-Star式のパーソナライズされた語りかけ
- **専門用語には必ず平易な説明を併記**
- **運気カーブ重ね表示**: 九星気学 × 六星占術 × 霊合合成を1チャートで比較
- **過去の振り返り**: 他サービスにない差別化ポイント
- **長期フォーキャスト**: 良い時・悪い時をフィーチャーし、具体的アクション提案

### Lesson Learned
- **九星気学計算エラー**: LLMが定数を10ではなく11で記憶していた（七赤金星→八白土星に誤算）
- **対策**: 占術ロジックはLLM生成禁止。OSSライブラリ使用 + ユニットテスト必須

### Market Research Summary
- 占い市場: 997億円（2023年度、矢野経済研究所）
- ユーザー層: 30-50代女性が90%。男性/ビジネス層は未開拓
- 競合: LINE占い、ココナラ占い、占いTV（月額980円）、Co-Star（米国500万DL）
- 差別化: 複数占術×心理学×ダッシュボード可視化の統合サービスはゼロ
- AI占い: ポラリス（エクサウィザーズ、占星術単体）が唯一の類似サービス
- 価格帯: 占いサブスク月額400-980円、性格診断買い切り2,000-10,000円
- 法的注意: 景品表示法（「当たる」禁止）、医師法（健康アドバイスの範囲）

### Risks
- 占術の科学的根拠問題 → 「エンターテインメント＋自己理解ツール」のポジショニング
- ライセンス制約（MBTI/SF） → 独自フレームワーク
- ターゲット不明瞭（占い好き vs ビジネス層） → Phase 0でまず自分用に作り、PMF検証
